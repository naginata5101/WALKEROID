こちらは「WALKEROID」というゲームです。

1.はじめに
　何かと残念な代物が出来てしまいました…。それがこれです。技術がないのに時間を割かなかった結果がこれだよ！
　本当に素材提供してくれた本田君（音）と新田君（絵）に申し訳ないです。
　こんなゲームですが、プレイして頂ければ光栄です。無理して楽しいフリをしなくても大丈夫です。
　内容の稚拙さを隠すために難易度を高くしました。結果、難易度は何も隠さず、ゲームバランスがひどくなっただけでした。
　このゲームをクリアして下さる奇特な方をお待ちしております。

2.操作方法
　カーソルキー…自機（ミク）の移動
　Zキー・・・・ 自機のショット

　Xキー・・・・ 強制終了（チートモードのみ）


　この他は画面の指示に従って下さい。

3.ゲームについて
　自機の弾は「壁」と「敵」を消し（倒し）ます。「敵の弾」と「穴」は消せません。

　「敵」は弾を撃ってきます。勿論、当たるとダメージです。しかし「壁」は撃ちません。要は弾を撃たない敵です。
　敵の弾は3way、対して自機の弾は1way。しかも敵は多数。圧倒的不利です。
　ですので、敵を倒すことは忘れて逃げ回るのも手です。というかむしろ正攻法です。
　敵は画面内に現れて少しすると弾を撃ちます。なので、敵が弾を撃つ前に倒せるとかなり楽になります。「撃たれる前に撃て」です。
　上記の理由から、画面の上の方には一切弾が来ません。だから、穴に注意して上避けに徹するのも手です。（ただし得点が残念なことに…。）

　タイトル画面で「ｴﾀｰﾅﾙ･ﾌｫｰｽ･穴 / 落ちると死ぬ」と書いていますが、ただの穴です。ですが落ちると即死です。このゲーム最大の脅威です。
　穴は当たり判定が小さくなっています。けれど、無理して落ちたら即死ですので十分気をつけてください。
　因みに、ダメージモーション中 及び ダメージ後の無敵時間中 でも容赦なく落ちます。モーション中に落ちないよう御注意を。

　このゲームでは、ダメージ後の無敵時間中は弾を撃てません。ですので、一度被弾すると状況がかなり悪化します。（連続で被弾なんてよくあること）
　なので、無敵時間中にできること、と言ったら強行突破ぐらいですけど、この際に穴に飛び込むミクが続出！（すると思う…）
　ミクさんは足が速いので、簡単に飛び込みます。点滅中の移動は危険ですので十分注意しましょう。

　アイテムの葱（少・多）は「敵」の撃破数に応じて出現します。「壁」の撃破数は関係ありません。
　一応、葱（多）なんて作りましたけど、出現させるのは比較的無理です。こんな避けゲーでは条件が難し過ぎます。だから太っ腹の500点。
　はちゅねさん(1up)は基本釣りです、無視しましょう。無理して取ろうとしたら、ダメージ受けたり、最悪穴に落ちます。（楽に取れるのもあるけど。）
